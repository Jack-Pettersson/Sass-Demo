Hello, SASS!

This is a small HTML + SASS project.

"Sass makes CSS fun and powerful! With variables, nesting, and more, you can write cleaner and more maintainable styles."

Sass (Syntactically Awesome Style Sheets) is a preprocessor scripting language that is interpreted or compiled into CSS. It helps you write more flexible and organized stylesheets.
